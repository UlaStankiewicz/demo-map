Demo project by Tomasz Babiak for recruitment processes by Bright Inventions

How to run the server

1. Go to `server-app` folder

### `cd .\server-app\`

2. Run script

### `npm start`

How to run the application

1. Go to `bright-inventions-app` folder

### `cd .\bright-inventions-app\`

2. Run script and wait until browser will open. If it'll not open - open it and type `http://localhost:3000`

### `npm start`
